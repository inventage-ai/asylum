# version

Show Asylum version information.

## Usage

```
asylum version
asylum version --short
asylum --version          # flag alias
asylum --version --short  # flag alias
```

## Description

Prints the Asylum version string and exits.

## Flags

| Flag | Description |
|------|-------------|
| `--short` | Print only the version number without the `asylum` prefix |

## Examples

```sh
$ asylum version
asylum 0.5.0 (abc1234)

$ asylum version --short
0.5.0 (abc1234)
```
